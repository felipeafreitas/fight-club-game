## ABOUT THE GAME

This is a 1x1 fighting game. Rules are simples: wins the best one in a two-out-of-three fight series.

## KEYBOARD INSTRUCTIONS

To control Tyler:
- Move left: z
- Move Right: x
- Jump: a
- Punch: s

To control the Narrator: 
- Move left: Left Arrow
- Move Right: Right Arror
- Jump: Up Arrow
- Punch: Down Arrow
